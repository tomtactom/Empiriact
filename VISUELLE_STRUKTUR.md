## 🎨 Design-System

```

└─ Symbolisiert: Grundlagen/Basis (tiefe Farbe)

Modul 2: Denkstile verstehen
Violett: #8B5CF6 (RGB: 139, 92, 246)
└─ Symbolisiert: Übergang/Unterscheidung (Mischfarbe)

Modul 3: RND verstehen & regulieren
Blau: #3B82F6 (RGB: 59, 130, 246)
└─ Symbolisiert: Lösungsorientierung (helle, aktive Farbe)
```

### Icons
```

└─ Einheitlichkeit & Erkennbarkeit
```

```

```

## 🔀 Inhalts-Flow

```
Grübeln & Rumination
├─ Was ist Grübeln?
│  ├─ Definition (Wiederkäuen)
│  ├─ Normale Ausprägungen
│  └─ Häufige Themen
│
├─ Warum ist es problematisch?
│  ├─ Emotionale Konsequenzen
│  ├─ Körperliche Auswirkungen
│  ├─ Soziale Probleme
│  └─ Der Verstärkungskreislauf
│
└─ Erkenntnis: "Das ist normal, aber trainierbar"
   (→ Motivation für Modul 3)

            ↓ (Progression)

Denkstile verstehen
├─ Ungünstige Denkstile (wie Grübeln)
│  ├─ Charakteristika
│  ├─ Probleme
│  └─ Wie sie entstehen
│
├─ Günstige Denkstile (konstruktiv)
│  ├─ Charakteristika
│  ├─ Vorteile
│  └─ Was sie unterscheidet
│
└─ Erkenntnis: "Ich kann unterscheiden"
   (→ Selbst-Beobachtung möglich)

            ↓ (Progression)

RND verstehen & regulieren
├─ Tieferes Verständnis
│  ├─ Was ist RND?
│  ├─ Verschiedene Formen
│  └─ Der gemeinsame Kern
│
├─ Die Hoffnungsbotschaft
│  ├─ RND ist erlernt
│  ├─ RND ist nicht die Persönlichkeit
│  └─ RND ist veränderbar
│
├─ Konkrete Strategien
│  ├─ Erkennung (wann grübel ich?)
│  ├─ Unterbrechung
│  └─ Neue Gewohnheiten
│
└─ Erkenntnis: "Ich kann mich ändern"
   (→ Handlungsfähigkeit & Hoffnung)
```
